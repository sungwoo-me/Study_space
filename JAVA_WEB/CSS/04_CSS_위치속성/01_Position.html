1. 정적 위치 지정 방식

정적 위치 지정은 top, right, bottom, left와 같은 속성 영향을 받지 않는다.

단순히 휍 페이지에 흐름에 따라 차례대로 요소들을 위치 시킨다.

2. 상대 위치 지정 방식

상대 위치 지정 방식은 해당 HTML 요소의 기본 위치를 기준으로 위치를 설정 하는 방식이다.

3. 고정 위치 지정 방식

고정 위치 지정 방식은 뷰포트를 기준으로 위치를 설정하는 방식이다.

4. 절대 위치 지정 방식

절대 위치 지정방식은 고정위치와 비슷하지만 뷰포트가 아닌 설정된 조상요소를 기준으로 위치를 설정한다.
