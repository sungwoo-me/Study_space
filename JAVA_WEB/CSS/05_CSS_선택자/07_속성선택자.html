2. 문자열 속성 선택자

- [속성이름~="속성값"] 선택자
-> 띄어쓰기를 기준으로 하나의 단어를 포함하는 요소 선택
- [속성이름|="속성값"] 선택자
-> 속성값이 정확히 하나의 단어이거나 -으로 시작하는 요소만 선택
- [속성이름^="속성값"] 선택자
-> 특정 속성값으로 시작되면 모두 선택된다.
- [속성이름$="속성값"] 선택자
-> 특정 속성값으로 끝나면 선택된다.
- [속성이름*="속성값"] 선택자
-> 특정 속성값으로 시작되면 모두 선택한다.
