1. 블록 요소

display 속성값이 블록인 요소는 새로운 라인에서 시작하며 해당 모든 너비를 차지한다.

p , div, h ul ol form 등 요소는 대표적인 블록 요소 입니다.

2. div 요소

div 요소는 다른 HTML요소들을 하나로 묶는데 사용되는 대표적인 블록 요소입니다.

div 요소는 여러 요소들의 스타일을 한번에 적용하기 위해 사용된다.

div요소를 이용한 스타일 적용

이렇게 div요소로 여러 요소들을 묶은 다음에 style 속성과 클래스 등을 이용하여 한 번에 스타일을 적용할 수 있습니다.

3. 인라인 요소

display 속성값이 인라인인 요소는 새로운 라인에서 시작하지 않습니다.

해당 HTML 요소의 내용 만큼만 차지한다.

span, a ,img 등 요소는 대표적인 인라인 요소입니다.

4. span 요소

문장중에 일부분만 따로 적용이 가능하다.
