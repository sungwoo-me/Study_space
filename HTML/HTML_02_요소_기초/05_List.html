1. Unordered list (ul 태그) : 순서가 없는 리스트

  • 1번쨰 요소
  • 2번째 요소
  • 3번째 요소

2. ul 태그 : 마커 모형 변형

  • 흰색 작은 원 모양
  • 사각형 모양

3. ol 태그 : 순서가 있는 리스트

  1. 첫번째 요소
  2. 두번째 요소
  3. 세번재 요소

4. ol 태그 : 마커 모형 변형

  1. 영어 대문자
  1. 영어 소문자
  1. 로마 숫자 대문자
  1. 로마 숫자 소문자

5. dl 태그 : 정의 리스트

부제목은 dt에 작성
실제 목록은 dd에 작성
